# Smart Process Optimizer

Aplicación web desarrollada como parte del proyecto APP1 del curso Diseño e Implementación de Sistemas de Información.

## Descripción

Smart Process Optimizer es una aplicación que permite analizar procesos mediante lógica basada en inteligencia artificial simulada.

El sistema evalúa la eficiencia de un proceso en función del número de pasos y el tiempo requerido, generando recomendaciones automáticas.

## Funcionalidades

- Ingreso de procesos
- Evaluación de eficiencia
- Generación de recomendaciones
- Visualización de resultados

## Tecnologías utilizadas

- HTML
- CSS
- JavaScript

## Lógica del sistema

La aplicación utiliza reglas simples para simular análisis inteligente:

- Muchos pasos + alto tiempo → proceso ineficiente
- Pasos moderados → mejora posible
- Pocos pasos → proceso eficiente

## Pruebas realizadas

Se realizaron pruebas funcionales:

| Caso | Pasos | Tiempo | Resultado |
|------|------|--------|----------|
| 1 | 8 | 20 | Ineficiente |
| 2 | 6 | 10 | Mejorable |
| 3 | 3 | 5 | Eficiente |

## Resultado

La aplicación cumple con los objetivos del proyecto, demostrando la aplicación de conceptos de AI Product Ops y Process Redesign.
